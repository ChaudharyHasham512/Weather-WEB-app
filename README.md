🌤️ Weather App

A simple and responsive weather application that allows users to check real-time weather conditions for any city in the world. Built using HTML, CSS, and JavaScript, this project demonstrates API integration and dynamic UI updates.

🚀 Features

🌍 Search weather by city name
🌡️ Displays temperature in real-time
💧 Shows humidity levels
🌦️ Weather condition indicators (☀️ Sunny, 🌧️ Rainy, ☁️ Cloudy)
⚠️ Error handling for invalid or empty inputs
📍 Get weather using your current location (Geolocation API)
🎨 Dynamic background based on weather conditions (optional enhancement)

🧠 Concepts Used

Fetching data from APIs using fetch()
Working with JSON data
DOM manipulation
Event handling in JavaScript
Error handling
Geolocation API

HTML5 – Structure
